# Install

Execute this commands:

```bash
wget -O /usr/bin/htop https://raw.githubusercontent.com/mr-manuel/venus-os_helpful-scripts/master/htop/armv7/htop
wget -O /lib/libncursesw.so.5 https://raw.githubusercontent.com/mr-manuel/venus-os_helpful-scripts/master/htop/armv7/libncursesw.so.5
wget -O /lib/libtinfo.so.5 https://raw.githubusercontent.com/mr-manuel/venus-os_helpful-scripts/master/htop/armv7/libtinfo.so.5
chmod +x /usr/bin/htop
chmod +x /lib/libncursesw.so.5
chmod +x /lib/libtinfo.so.5
```
